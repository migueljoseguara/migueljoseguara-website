# migueljoseguara-website
## Step 1: Searching Examples and Structuring Benchmarks
The first step of this project was searching for examples that are similiar or have elements of what I want to build and structuring benchmarks from these examples.
Searching for these examples I concluded that the appearance of the website would be inspired by portuguese tiles -- a simplified approach to the designs portrayed on those tiles. Along them, the icons used in the 3DS main menu were an inspiration as well. At this stage, I had a pretty good mental image of what I wanted to build.
